# 데이터베이스 모니터링

모든 캠페인 인스턴스의 자산, 워크플로우 및 데이터는 데이터베이스에 저장됩니다. 데이터베이스가 최대 용량에 도달하면 인스턴스에 문제가 발생할 수 있습니다(로그인 불가, 이메일 전송 등). 따라서 최적의 성능을 보장하기 위해서는 인스턴스 데이터베이스를 정기적으로 모니터링하는 것이 좋습니다.
